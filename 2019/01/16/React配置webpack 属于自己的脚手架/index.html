快速创建webpack@4+项目

  • 项目初始化

    • npm init -y

  • 创建目录和文件

    • 创建项目根目录 (源代码目录) src

    • 创建产品目录 dist

    • 在 src 目录下创建 index.html

      • 入口文件 index.js

    • 执行命令 npm i webpack webpack-cli -D

    • 创建 webpack配置对象 webpack.config.js

      • 注意 : webpack@4+ 默认入口文件是 index.js 出口文件 是 main.js

    • 安装

      • 执行命令 npm i webpack-dev-server -D

    • 修改 package.json文件

    • 安装

      • 执行命令 npm i html-webpack-plugin -D
